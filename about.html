Our Story

Pear Badminton Club was founded on a simple philosophy: sports spaces should be both high-performance and beautifully designed. Inspired by family-run sports initiatives across the globe, we sought to build an inviting, aesthetic club environment where casual players and competitive athletes alike can find their perfect match.

1. Inspiration & Design Logic

The operational flow of this platform is inspired by a pickleball club startup in Thailand, adapting real-world scheduling demands into a frontend web application. The design uses an organic earth-tone palette (#2c3e25, #87a96b, #f7f5f0) to evoke a premium lifestyle brand vibe.

2. Technical Specifications

Built using semantic HTML5 structures, customized layouts via CSS Grid and Flexbox, and state-driven JavaScript. The core programming highlight is a fully interactive reservation map engine powered by multi-state data arrays that recalculate court availability on user request.
